Esta aplicación

NewEgg WS Getter es una aplicación realizada en lenguaje Python, debido a la mayor accesibilidad de las librerías necesarias para nuestro agente inteligente con Web Scraping, utilizando además el framework Django para la creación de una interfaz web básica con soporte del lenguaje antes mencionado para el frontend (UI) y backend (navegación y obtención de datos)

Página web scrapeada: NewEgg

Features importantes

  • Extracción de información de las cartillas de la página web con información de productos (nombre, precio, en venta, imagen y link)
  • Guardado de información scrapeada en archivo JSON para extracción posterior
  • Búsqueda de información guardada del scraping con un searchbar
